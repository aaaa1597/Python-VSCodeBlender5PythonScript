# Blenderで3DCGアニメを作る。
```
アルファベット3文字TKSを使って起承転結の展開のわかる物語性のある15秒程度の3DCGアニメーションをBlenerで実行するpythonスクリプトを作ってほしいのだけど、まずシナリオというか、話の展開というかいろんなパターンを考えたい。まず"起"の部分の展開を20パターン上げてくれる？
```
了解！
3文字 T / K / S が主役の “起承転結のある15秒3DCGアニメーション” 用に、
まず 「起」だけの展開アイデアを20パターン 用意したよ。
どれも後で Python＋Blender で実装しやすいよう、シンプルだけど絵になる始まりにしてある。
「承」「転」「結」をそれぞれ20案、お願いしまっす

## 🎬 アルファベットアニメ
### 出現パターン
- 上から落ちてくる。
- 下からせりあがってくる。
- 霧や煙からフェードイン
### スライド系
- 横からスライドイン
- 慣性付きスライド
- 回転しながらスライド
### 回転変形
- 3D回転で現れる
- スケールアップ
- 文字がバラバラに分解→再構築
### カメラワーク連動
- カメラがパンして文字が追従: 視点移動と同期
- カメラがズームして文字が迫る: 迫力演出
### 物理演出
- バウンドしながら着地
- 風で流されるように移動
- 動力 + 慣性で転がる
### 特殊効果
- 光ラインで描かれる
- 液体っぽく変形して固まる
- 炎や電流で文字が浮かび上がる
### さらに
- 「T」「K」「S」が順番に動く
- 同期して動く

import bpy

# 既存のデフォルトオブジェクトを削除
bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete(use_global=False)

# テキストオブジェクト \"T\" を作成
bpy.ops.object.text_add(location=(-3, 0, 0))
text_t = bpy.context.active_object
text_t.data.body = \"T\"
text_t.name = \"Text_T\"

# テキストに厚みを追加
text_t.data.extrude = 0.3
text_t.data.bevel_depth = 0.02

# テキストオブジェクト \"K\" を作成
bpy.ops.object.text_add(location=(0, 0, 0))
text_k = bpy.context.active_object
text_k.data.body = \"K\"
text_k.name = \"Text_K\"

# テキストに厚みを追加
text_k.data.extrude = 0.3
text_k.data.bevel_depth = 0.02

# テキストオブジェクト \"S\" を作成
bpy.ops.object.text_add(location=(3, 0, 0))

text_s.name = \"Text_S\"

# テキストに厚みを追加
